Introduction

Python is an easy-to-learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming.


Python's elegant syntax and dynamic typing make it an ideal language for scripting and rapid application development in many areas on most platforms.

Hello World

To get started with Python, open a text editor and write your first "Hello World" Python code!

print("Hello World")

Run the code and watch the magic unfold!

Variables

You use variables as symbolic names for values in your program. The names of variables, called identifiers, conform to certain rules.


A Python identifier must be unique and start with a character (A-Z/a-z). It cannot include any special characters such as * or ! either. They are also case-sensitive.

Declaring Variables

Python makes declaring variables very easy. For example:

a = 10 #this declares an integer b = "Hello World!" #this declares a string c = 9.0 #this declares a floating point d = True #this declares a boolean
Data Types

The built-in data types in Python include:

  • str
  • int, float, complex
  • list, tuple, range
  • dict
  • set, frozenset
  • bool
  • bytes, bytearray, memoryview
Variable Scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the document. When you declare a variable within a function, it is called a local variable, because it is only available within that function.

Function Declarations

A function definition consists of the function keyword, a list of arguments to the function, enclosed in parentheses, and separated by commas.
